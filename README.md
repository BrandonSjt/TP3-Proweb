# TP3 - Profil Pribadi

Halaman web statis sederhana untuk menampilkan profil pribadi, meliputi foto, video perkenalan, deskripsi singkat, tabel keterampilan, form kontak, dan tautan media sosial.

## Ringkasan Proyek

Project ini dibuat menggunakan HTML dan CSS internal (tanpa framework). Seluruh tampilan dan struktur halaman berada dalam satu file utama.

## Fitur

- Menampilkan judul dan identitas singkat.
- Menampilkan foto profil.
- Menampilkan video perkenalan.
- Menampilkan bagian "Tentang Saya".
- Menampilkan tabel keterampilan.
- Menyediakan form "Hubungi Saya" (nama, email, pesan).
- Menyediakan tautan ke media sosial.

## Struktur Folder

- TP3.html: Halaman utama.
- FotoProfil.jpeg.jpg: Gambar profil yang ditampilkan pada halaman.
- Oh my gah!.mp4: Video perkenalan yang diputar pada halaman.

## Cara Menjalankan

1. Pastikan semua file berada dalam folder yang sama.
2. Buka file TP3.html menggunakan browser.
3. Halaman profil akan tampil beserta foto dan video.

## Teknologi yang Digunakan

- HTML5
- CSS3 (internal style)

## Catatan

- Form kontak saat ini hanya tampilan (belum terhubung ke backend).
- Jika foto atau video tidak muncul, periksa nama file dan lokasi file sudah sesuai.
- Untuk keamanan, saat membuka tautan eksternal dengan tab baru, disarankan menambahkan atribut rel="noopener noreferrer" pada setiap tag link yang memakai target="_blank".

## Pengembangan Lanjutan (Opsional)

- Menambahkan JavaScript untuk validasi form.
- Menghubungkan form ke backend (misalnya PHP, Node.js, atau layanan form pihak ketiga).
- Memisahkan CSS ke file terpisah agar struktur project lebih rapi.
- Menambahkan desain responsif yang lebih detail untuk berbagai ukuran layar.
